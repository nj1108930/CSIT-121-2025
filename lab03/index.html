Learning Web Standards

Labs 2 & 3

Published on March 2, 2025 by Isaac Njoroge

First Reflection - Lab02

Creating my first webpage was a challenge. At first, learning about HTML structure and web standards felt overwhelming, but breaking it down into smaller steps helped me understand it better. Now, I feel more confident in structuring HTML documents and following best practices for web development.

Second Reflection - Lab03

Applying CSS to my webpage helped me see how styling makes a huge difference in presentation. I learned how to use classes and IDs to target specific elements, making it easier to control the design. Understanding external stylesheets also showed me how to separate content from design, which improves organization. I faced some challenges with selectors at first, but after experimenting, I was able to fix issues and get my page looking better.

What I Learned

  1. How to structure a basic HTML document.
  2. The importance of semantic elements.
  3. How to create lists, links, and headings properly.
  4. How CSS enhances visual design.
  5. Using classes and IDs to style elements.
